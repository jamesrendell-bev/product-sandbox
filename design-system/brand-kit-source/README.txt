How to start a new deck

  1. Open PowerPoint. Set slide ratio to Widescreen 16:9 (1920 × 1080).
  2. For the cover: Design > Format Background > Picture or Texture
     Fill > insert BEV-Background-Title.png.
  3. For each content slide: alternate the gradient and white background
     PNGs the same way. Cover stays gradient/title; content alternates.
  4. Keep content out of the top-right (the lockup zone) and, on the
     cover, out of the bottom 1.5 inches (the client logo strip).
  5. Read the .md style guide for typography, spacing, voice rules.


Non-negotiables

  - Avenir Next throughout. Named faces only — never the Bold toggle.
  - BEV Navy #000615 only as the dark stop of the gradient.
  - Hot Pink #FF6CAA for emphasis only — never body text.
  - All cell text in tables vertically centred.
  - All left-aligned text shares the same left edge (0.5" from edge).
  - CERA® always with the registered mark.
  - BirdsEyeView is one word. Never "Birds Eye View".
  - Never use the Bold toggle on Avenir Next.
